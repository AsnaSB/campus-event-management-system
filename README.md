# Campus Event Management System

A Django-based Campus Event Management System developed using ShaktiDB.

## Project Overview

This project is designed to manage campus events efficiently by providing a centralized platform for:

- Club Management
- Event Management
- Participant Registration
- Attendance Tracking
- Certificate Generation

## Tech Stack

- Python
- Django
- ShaktiDB
- HTML
- CSS
- Bootstrap
